# ESP-32 调试过程中遇到的问题

1. 烧录问题

USB口接ESP-32c3可以直接用串口烧录

ESP32-C3 内部集成了 **USB OTG 控制器**，可以模拟出一个串口设备（CDC），实现以下功能：

- 直接通过板载 USB 口连接电脑，电脑会识别出一个虚拟 COM 口
- 支持下载固件（烧录）、串口调试输出，和普通 USB 转串口模块效果一致
- 部分开发板还支持通过这个 USB 口供电，不用额外电源

烧录的时候如果检测到com口 但是无法打开可能的原因：

有多条 idf_monitor 进程同时占用了 COM10。要把这些监视器进程全部结束，再立即重试烧录。
